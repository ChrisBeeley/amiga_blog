AMOS Professional⌗

Getting AMOS Professional working is very easy. Go to the link above and download the zip file where indicated. Extract it, and you just want the folder called “AMOS”. Copy that folder somewhere you can keep it, then just drop it in as a hard drive in FS-UAE (go to the hard drive tab, select the little picture of a folder next to an empty hard drive slot, and then select the AMOS folder). You’ll need to have a Workbench HD image too. That’s another blog post. I learned the above in these excellent blog posts but the advice is in a video and I prefer to read stuff myself.

My next step is to work through the tutorials in the previous link which culminates in… yes… Space Invaders. I’m not going to count that as a “win” for this step though, I’ll make my own. Good start though.
